# [POST MAIL](https://post-mail-fast.herokuapp.com/)
It is a mail posting web app inspired by mailchimp and build for flipr hackathon 9.0 



<h1><FEAURES/h1>

* Login page
* register page
* create mail page
  
* running page
* history page
